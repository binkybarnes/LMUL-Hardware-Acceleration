WHAT YOU NEED TO DO
================================================================================

The framework is ready, but you need to install gem5 and run it.

🐳 DOCKER INSTALLATION (RECOMMENDED FOR macOS)
--------------------------------------------------------------------------------

STEP 1: Read Docker Setup Guide (5 minutes)
  $ cd /workspaces/LMUL-Hardware-Acceleration/gem5
  $ cat DOCKER_SETUP.md           # Read this first! ⭐

STEP 2: Start Your Dev Container
  # In VS Code: Reopen in Container
  # You should already be in the container!

STEP 3: Install gem5 Dependencies (Inside Container - 2 minutes)
  $ apt-get update && apt-get install -y \
      m4 scons zlib1g-dev libprotobuf-dev protobuf-compiler \
      libgoogle-perftools-dev libboost-all-dev gcc-arm-linux-gnueabi

STEP 4: Clone & Build gem5 (Inside Container - 20-30 minutes)
  $ cd /workspace
  $ git clone https://github.com/gem5/gem5.git
  $ cd gem5 && git checkout stable
  $ scons build/ARM/gem5.opt -j$(nproc)
  
  Note: This takes time - perfect for a coffee break! ☕
  Note: gem5 will be in /workspace/gem5 which persists on your Mac!

STEP 5: Install LMUL Model (Still in container - 5-10 minutes)
  $ cd /workspace
  $ ./gem5/scripts/install_model.sh

STEP 6: Build Benchmark (2 minutes)
  $ cd gem5/benchmarks/matrix_multiply
  $ make ARCH=arm

STEP 7: Run Test Simulation (3-5 minutes)
  $ cd /workspace/gem5
  $ ./scripts/run_simulation.sh --test

STEP 8: Analyze Results (2 minutes)
  $ python3 scripts/extract_stats.py m5out/stats.txt

STEP 9: Verify Everything (1 minute)
  $ ./scripts/verify_setup.sh

Total Time: ~45-60 minutes

TROUBLESHOOTING
================================================================================

gem5 won't build:
  → Check dependencies: ./scripts/verify_setup.sh
  → Reduce parallel jobs: scons -j2 instead of -j$(nproc)

Model not found:
  → Run: ./scripts/install_model.sh

Benchmark compilation fails:
  → Install cross-compiler: sudo apt install gcc-arm-linux-gnueabi

Simulation hangs:
  → Try smaller test first: --test
  → Check benchmark is statically linked and for ARM

No accelerator statistics:
  → Verify model installed: ls gem5/src/dev/lmul_accel/
  → Enable debug: add --debug-flags=LMulAccel

For detailed help:
  → Read GETTING_STARTED.md
  → Run ./scripts/verify_setup.sh
  → Check component READMEs
